天鳳AI自動対戦ソフト 使用方法

※ 本ソフトは GitHub の Release ページ からダウンロードしてください。

Tenhou ID の入力
「Tenhou ID」欄に、ご自身の天鳳IDを
IDの先頭から完全な形で 正確に入力してください。
設定の保存
入力後、必ず 「Save」ボタン をクリックして設定を保存してください。
保存しない場合、設定は反映されません。
対局条件の設定
以下の項目をお好みに合わせて変更してください。
人数（四麻）
ロビー
場（東 / 南）
赤ドラの有無
対局スピード
対局開始
設定完了後、「Start」ボタン をクリックすると、
AIによる自動対戦が開始されます。
🇺🇸 English Instructions
Tenhou AI Auto Play Software – User Guide

※ Please download this software from the GitHub Release page.

Enter Tenhou ID
Please enter your full Tenhou ID from the beginning
into the “Tenhou ID” field.
Save Settings
After entering your ID, make sure to click “Save”.
Settings will not take effect unless saved.
Configure Match Settings
Adjust the following options as needed:
Number of players (4-player)
Lobby
Match length (East / South)
Red fives (Aka dora)
Game speed
Start Playing
Once all settings are completed, click “Start”
to begin AI automatic gameplay.


天凤AI自动对战软件 使用方法
※ 请从 GitHub 的 Release 页面下载本软件。
1. 输入 Tenhou ID
在「Tenhou ID」栏中，输入您自己的天凤ID。
请从ID开头开始完整、准确地填写整个ID。
2. 保存设置
输入完成后，请务必点击「Save」按钮保存设置。
如果不保存，设置将不会生效。
3. 设置对局条件
根据需要修改以下项目：
人数（四人麻将）
大厅（Lobby）
场次（东风战 / 南风战）
是否使用赤宝牌
对局速度
4. 开始对局
设置完成后，点击「Start」按钮，AI将开始自动进行对战。
